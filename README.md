# KitchenStr

A modern front-end web application built with React and Vite. This application provides a responsive and dynamic user interface.

## Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kuldeep456789/KitchenStr.git
   cd KitchenStr
   ```

2. **Navigate to the project directory:**
   ```bash
   cd kitchenstr
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
